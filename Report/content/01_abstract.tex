\section{Abstract}

Understanding abstract data structures is a fundamental challenge for computer science students due to the lack of intuitive visualization. This project presents an interactive data structure visualization application developed in C++ to address this limitation.

The system supports real-time visualization of core data structures, including Singly Linked Lists, Heaps, AVL Trees, and Graphs with shortest path algorithms. It provides step-by-step animation of operations such as insertion, deletion, and traversal, enabling users to observe dynamic changes during execution.

To ensure smooth and responsive interaction, the system leverages graphical libraries and adopts a modular architecture that synchronizes algorithm execution with user interface rendering.

Overall, the application enhances conceptual understanding and demonstrates the effectiveness of interactive visualization as a learning tool for data structures and algorithms.
